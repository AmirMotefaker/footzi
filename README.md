# ⚽ Footzi - اپلیکیشن آنالیز فوتبال

یک اپلیکیشن وب فول-استک برای نمایش و آنالیز آمار بازیکنان لیگ برتر انگلیس. این پروژه با هدف ساخت یک سیستم کامل از جمع‌آوری داده تا نمایش به کاربر، با استفاده از تکنولوژی‌های مدرن توسعه داده شده است.



---

## ✨ ویژگی‌های کلیدی

* **جمع‌آوری خودکار داده (ETL):** دریافت آمار به‌روز بازیکنان از APIهای عمومی.
* **API قدرتمند:** بک‌اند ساخته شده با NestJS برای ارائه داده‌ها به صورت بهینه.
* **رابط کاربری مدرن:** فرانت‌اند تعاملی ساخته شده با Next.js و React.
* **صفحات داینامیک:** صفحه اختصاصی برای مشاهده جزئیات هر بازیکن.
* **جستجوی آنی:** قابلیت فیلتر و جستجوی بازیکنان در لیست.
* **صفحه‌بندی (Pagination):** نمایش لیست بازیکنان در صفحات متعدد برای تجربه کاربری بهتر.
* **وضعیت بارگذاری (Loading State):** نمایش اسکلت لودینگ برای بهبود تجربه بصری کاربر.

---

## 💻 استک فناوری (Tech Stack)

این پروژه به صورت یک **Monorepo** مدیریت می‌شود که شامل سه بخش اصلی است:

* **🖥️ فرانت‌اند (Frontend):**
    * **Next.js**
    * **React**
    * **TypeScript**
    * **Tailwind CSS**

* **🧠 بک‌اند (Backend - API):**
    * **NestJS**
    * **TypeScript**
    * **TypeORM**

* **📊 پایپ‌لاین داده (ETL):**
    * **Python**
    * **Prefect** (برای ارکستراسیون)
    * **Requests** (برای ارتباط با API)
    * **Pandas**

* **🗃️ دیتابیس (Database):**
    * **PostgreSQL**

## 🚀 راه‌اندازی و اجرا (Getting Started)

برای اجرای این پروژه به صورت محلی، مراحل زیر را دنبال کنید:

### ۱. پیش‌نیازها
* **Node.js** (نسخه 18 یا بالاتر)
* **Python** (نسخه 3.11 یا بالاتر)
* **PostgreSQL**
* **Git**

### ۲. کلون کردن پروژه
```bash
git clone [https://github.com/YourUsername/footzi.git](https://github.com/YourUsername/footzi.git)
cd footzi
```
*(به جای `YourUsername`، نام کاربری گیت‌هاب خود را قرار دهید.)*

### ۳. راه‌اندازی دیتابیس
1.  مطمئن شوید که سرویس PostgreSQL شما در حال اجراست.
2.  با استفاده از ابزاری مانند **pgAdmin**، یک دیتابیس جدید به نام `footzi_stats` بسازید.
3.  اسکریپت ساخت جداول را اجرا کنید: محتوای فایل `etl/infra/postgres/init.sql` را در Query Tool دیتابیس `footzi_stats` اجرا کنید تا جداول اولیه ساخته شوند.

### ۴. راه‌اندازی پایپ‌لاین داده (ETL)
این بخش داده‌های اولیه را جمع‌آوری کرده و در دیتابیس شما قرار می‌دهد.

```bash
# وارد پوشه etl شوید
cd etl

# محیط مجازی پایتون را بسازید و فعال کنید
python -m venv .venv
# For Windows PowerShell:
.venv\Scripts\activate

# پکیج‌های مورد نیاز را نصب کنید
pip install -r requirements.txt 
# (نکته: برای ساخت این فایل، می‌توانید از دستور pip freeze > requirements.txt استفاده کنید)

# Flow اصلی را برای دریافت داده‌ها اجرا کنید
python -m etl.flows.premier_league
```

### ۵. راه‌اندازی بک‌اند (API)
این سرور، داده‌ها را از دیتابیس خوانده و در اختیار فرانت‌اند قرار می‌دهد.

```bash
# از ریشه پروژه، وارد پوشه api شوید
cd api

# پکیج‌های npm را نصب کنید
npm install

# سرور را در حالت توسعه اجرا کنید
npm run start:dev
```
سرور API روی `http://localhost:3000` اجرا خواهد شد.

### ۶. راه‌اندازی فرانت‌اند
این بخش، رابط کاربری وب‌سایت را اجرا می‌کند.

```bash
# (در یک ترمینال جدید) از ریشه پروژه، وارد پوشه frontend شوید
cd frontend

# پکیج‌های npm را نصب کنید
npm install

# سرور توسعه را اجرا کنید
npm run dev
```
اپلیکیشن در مرورگر شما روی `http://localhost:3001` قابل مشاهده خواهد بود.

## 🔗 اندپوینت‌های API

* `GET /`: دریافت لیست تمام بازیکنان برتر.
* `GET /:id`: دریافت جزئیات یک بازیکن خاص بر اساس ID.
