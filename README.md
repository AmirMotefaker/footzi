# ⚽ Footzi - نگاهی عمیق به آمار فوتبال

**Footzi** یک پلتفرم وب مدرن و متن‌باز برای تحلیل دقیق و بصری آمار بازیکنان لیگ برتر انگلیس است. این پروژه به عنوان یک نمونه کار فول-استک، تمام چرخه حیات داده را از جمع‌آوری تا ارائه به کاربر نهایی پوشش می‌دهد.

---

## 💡 چرا Footzi متفاوت است؟

بر خلاف بسیاری از اپلیکیشن‌های آماری که تنها به نمایش داده‌های خام بسنده می‌کنند، Footzi با هدف ارائه **شفافیت و عمق بیشتر** ساخته شده است. تمرکز اصلی این پروژه بر موارد زیر است:

* **شفافیت داده (Data Transparency):** هر آمار ارائه شده قابلیت ردیابی تا منبع اصلی را دارد.
* **دسترسی آزاد (Open Access):** تمام کدها و فرآیندها به صورت متن‌باز در دسترس همگان قرار دارد.
* **معماری مدرن:** استفاده از یک استک فناوری به‌روز و مقیاس‌پذیر که نشان‌دهنده یک رویکرد حرفه‌ای در مهندسی نرم‌افزار است.

---

## ✨ ویژگی‌های کلیدی

* **جمع‌آوری خودکار داده (ETL):** پایپ‌لاین قدرتمند پایتون برای دریافت آمار به‌روز و جزئیات کامل بازیکنان.
* **API بهینه:** بک‌اند ساخته شده با NestJS برای ارائه سریع و امن داده‌ها.
* **رابط کاربری تعاملی:** فرانت‌اند پویا و سریع ساخته شده با Next.js و React.
* **صفحات داینامیک:** صفحه اختصاصی برای مشاهده جزئیات عمیق هر بازیکن.
* **جستجوی آنی و صفحه‌بندی:** تجربه کاربری روان برای مرور لیست بلند بازیکنان.
* **وضعیت بارگذاری (Loading State):** نمایش اسکلت‌های لودینگ برای حسی بهتر هنگام دریافت داده.

## 💻 استک فناوری (Tech Stack)

* **🖥️ فرانت‌اند:** Next.js, React, TypeScript, Tailwind CSS
* **🧠 بک‌اند (API):** NestJS, TypeScript, TypeORM
* **📊 پایپ‌لاین داده (ETL):** Python, Prefect
* **🗃️ دیتابیس:** PostgreSQL

## 🔗 اندپوینت‌های API

* `GET /`: دریافت لیست تمام بازیکنان برتر.
* `GET /:id`: دریافت جزئیات یک بازیکن خاص بر اساس ID.
